## Usage

    terraspace check_setup

## Description

Check setup.

## Example

    $ terraspace check_setup
    Detected Terrspace version: 0.3.3
    Detected Terraform bin: /home/ec2-user/.tfenv/bin/terraform
    Detected Terraform v0.13.2
    Terraspace requires Terraform v0.12.x and above
    You're all set!
    $

[--verbose], [--no-verbose]
