## Usage

    terraspace logs truncate

## Description

Truncates logs. IE: Keeps the files but removes contents and zero bytes the files.

## Example

    $ terraspace logs truncate
    Truncating log files in log/
    $


## Options

```
y, [--yes], [--no-yes]  # bypass are you sure prompt
```
