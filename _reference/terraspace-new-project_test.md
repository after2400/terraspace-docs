## Usage

    terraspace new project_test NAME

## Description

Generates new project test.

## Example

    $ terraspace new project_test demo
    => Creating project test: demo
           exist
          create  spec/fixtures/tfvars/demo.tfvars
          create  spec/stacks/demo/main_spec.rb
    $
